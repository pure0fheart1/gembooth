CMD & Powershell Cheat Sheet

File & Directory Management

Command Example Description Action
dir /b /s *.log Find all .log files recursively, showing bare path only.
dir /a:d List only directories. (/a:-d for only files).
cd C:\Users\ Change directory. Use cd .. to go up one level.
mkdir NewFolder Create a new directory (folder).
rmdir /s /q OldFolder Remove a directory and its contents quietly (no prompt).
xcopy Source C:\Dest /s /e /i Copy directories and subdirectories, creating Dest if it doesn't exist.
robocopy C:\Source D:\Dest /e /z /mir Mirror a directory (robust, restartable, deletes extra files in Dest).
ren old.txt new.txt Rename a file. (rename)
del /f /q file.txt Delete a file forcefully and quietly.
echo "Hello" > new.txt Create a new file with text (overwrites).
echo "More" >> new.txt Append text to an existing file.
mklink /D \MyLink C:\TargetFolder Create a symbolic link (directory link). (Run as Admin)
icacls C:\Folder /grant "MyUser":(OI)(CI)F Grant a user Full Control to a folder and sub-items. (Run as Admin)
icacls C:\Folder /remove "MyUser" Remove all permissions for a user from a folder. (Run as Admin)
takeown /f C:\Folder /r /d y Take ownership of a folder recursively. (Run as Admin)

Text Manipulation

Command Example Description Action
findstr /i /c:"login failed" *.log Find text (case-insensitive) in multiple files.
type log.txt | more Display file contents one page at a time.

Network

Command Example Description Action
ipconfig /all Display all network adapter information.
ipconfig /flushdns Clear the local DNS cache.
ping google.com -t Test connectivity continuously until stopped (Ctrl+C).
tracert google.com Trace the network route to a host.
netstat -anb Show connections and the executables using them. (Run as Admin)
nslookup google.com Query DNS for IP address information.
arp -a Display the local ARP cache (IP to MAC address mapping).
route print -4 Display the IPv4 routing table.
net use Z: \\Server\Share /persistent:yes Map a network drive (Z:) to a share, making it persistent.
net share MyShare=C:\Folder /grant:Everyone,READ Share a folder with a specific name and permissions.
netsh wlan show profiles List all saved Wi-Fi profiles on the system.
netsh wlan show profile name="MyWiFi" key=clear Show the password for a specific Wi-Fi profile. (Run as Admin)
netsh advfirewall firewall add rule name="Block-App" dir=out program="C:\app.exe" action=block Block a specific application from accessing the network. (Run as Admin)

System & Process Management

Command Example Description Action
tasklist | find "chrome" Find a specific process.
taskkill /F /IM notepad.exe Forcefully terminate a process by its image name.
sc query Spooler Query the status of a service (e.g., Print Spooler).
sc stop Spooler Stop a service. (Run as Admin)
sc config "MyService" start= delayed-auto Change a service's start type (e.g., to delayed auto-start).
schtasks /query /fo LIST /v List all scheduled tasks in a verbose list format.
schtasks /create /tn "MyTask" /tr "notepad.exe" /sc ONCE /st 23:00 Create a new task to run notepad at 11:00 PM tonight.
sfc /scannow System File Checker: Scans and repairs system files. (Run as Admin)
chkdsk C: /f Check disk for errors and fix them. (Run as Admin)
shutdown /r /t 0 Restart the computer immediately.
gpupdate /force Force an update of Group Policy settings.
wmic product get name,version List installed software (can be slow). (Run as Admin)
wmic bios get serialnumber Get the computer's BIOS serial number. (Run as Admin)
driverquery /v List all installed device drivers (verbose).
powercfg /energy Generate a power efficiency report. (Run as Admin)
powercfg /lastwake Show what device or event last woke the PC from sleep.

User Management (Run as Admin)

Command Example Description Action
net user NewUser Pa$$w0rd /add Create a new local user.
net user OldUser /delete Delete a local user.
net localgroup Administrators NewUser /add Add a user to the local Administrators group.
net localgroup "MyGroup" /add Create a new local group.

Registry Management (Run as Admin)

Command Example Description Action
reg query HKLM\Software\Microsoft\Windows Query a registry key for its subkeys and values.
reg add HKCU\Software\MyApp /v Version /t REG_SZ /d 1.0 Add or update a registry value.
reg delete HKCU\Software\MyApp /f Force delete a registry key and all its values.
reg export HKCU\Software\MyApp C:\Backup.reg Export a registry key to a .reg file.
reg import C:\Backup.reg Import a .reg file into the registry.

Disk Management (Run as Admin)

Command Example Description Action
diskpart Enter the interactive disk partitioning tool. (Sub-commands below)
(echo list disk) | diskpart List all disks (non-interactive example).
(echo select disk 1 & echo clean) | diskpart Select disk 1 and erase it. **DANGEROUS!**
format E: /fs:NTFS /q /v:MyData Quick format drive E: with NTFS and set label to "MyData".
label E: NewLabel Change the label of drive E:.

Batch Scripting Logic

Command Example Description Action
@echo off Turn off command echoing in a script.
set MY_VAR="Hello World" Set an environment variable for the session.
echo %MY_VAR% Use (echo) a variable.
if exist "file.txt" (echo "File exists!") Check if a file exists.
if %ERRORLEVEL% NEQ 0 (echo "Command failed!") Check if the previous command failed.
for /f "tokens=1,2" %a in (data.txt) do echo %a %b Loop through each line of a file, splitting it by spaces.
call :myFunction Call a subroutine (function) within a batch file.

File & Directory Management

Command Example Description Action
Get-ChildItem -Recurse -Filter "*.log" -File Find all .log files recursively. (Alias: gci, ls, dir)
Set-Location C:\Users\ Change directory. (Alias: sl, cd)
New-Item -Path . -Name "NewFolder" -ItemType Directory Create a new directory.
Remove-Item -Path .\OldFolder -Recurse -Force Remove a non-empty folder forcefully. (Alias: ri, rm)
Copy-Item -Path C:\Source -Destination D:\Dest -Recurse Copy an entire directory recursively. (Alias: cpi)
Rename-Item -Path .\old.txt -NewName new.txt Rename an item. (Alias: rni, ren)
Get-Content -Path .\log.txt -Tail 10 Get the last 10 lines of a file. (Alias: gc)
"Hello" | Out-File -FilePath new.txt -Encoding utf8 Create a new file with text (UTF-8 encoding).
Add-Content -Path .\log.txt -Value "New entry" Append text to an existing file (safer than Out-File -Append).
Compress-Archive -Path .\MyFolder -DestinationPath .\Archive.zip Create a .zip file from a folder.
Expand-Archive -Path .\Archive.zip -DestinationPath .\Unzipped Extract a .zip file.
Get-FileHash .\myfile.txt -Algorithm SHA256 Calculate the hash (checksum) of a file.
Get-Acl -Path C:\Folder | Format-List Get the security permissions (ACLs) for a folder.
$acl = Get-Acl C:\Folder; $acl.SetOwner([System.Security.Principal.NTAccount]"Administrators") Set the owner of a folder (requires 2 steps). (Run as Admin)

Text Manipulation & Data

Command Example Description Action
Select-String -Path .\*.log -Pattern "error" Find text in multiple files. (Alias: sls)
(gc file.txt) -replace "old", "new" | Out-File new.txt Find and replace text in a file, saving to a new file.
Import-Csv -Path .\data.csv Read a CSV file and convert it to objects. (Alias: ipcsv)
Get-Process | Export-Csv -Path proc.csv -NoTypeInformation Export command output to a CSV file. (Alias: epcsv)
Get-Content .\data.json | ConvertFrom-Json Convert JSON text to a Powershell object.
$myObject | ConvertTo-Json Convert a Powershell object to a JSON string.
[xml]$doc = Get-Content .\data.xml; $doc.SelectNodes("/items/item") Read and parse an XML file, then query it with XPath.

Object Manipulation (The Pipeline)

Command Example Description Action
Get-Process | Where-Object { $_.CPU -gt 100 } Filter objects based on a property. (Alias: where, ?)
Get-Process | Select-Object -Property Name, ID, CPU Select only specific properties from objects. (Alias: select)
Get-Process | Sort-Object -Property CPU -Descending Sort objects by a property. (Alias: sort)
Get-Service | Group-Object -Property Status Group objects based on a common property value. (Alias: group)
Get-Process | Measure-Object -Property WorkingSet -Sum -Average Calculate statistics (sum, avg, etc.) on object properties. (Alias: measure)
Get-Process | Select-Object -First 5 Get the first 5 objects from the pipeline.
Get-Process | Out-GridView Send pipeline output to an interactive, filterable grid window.

Network

Command Example Description Action
Get-NetIPConfiguration Get detailed IP configuration (replaces ipconfig).
Get-NetConnectionProfile Show network profile (Public, Private, Domain).
Set-NetConnectionProfile -Name "Ethernet" -NetworkCategory Private Change a network's profile from Public to Private. (Run as Admin)
Clear-DnsClientCache Clear the local DNS cache.
Test-Connection google.com -Count 2 Ping a host 2 times. (replaces ping)
Test-NetConnection google.com -Port 443 Test if a specific port is open on a remote host.
Get-NetTCPConnection -State Listen List all listening ports (replaces netstat -an).
Resolve-DnsName google.com Query DNS information (replaces nslookup).
New-NetFirewallRule -DisplayName "Block-App" -Direction Outbound -Program "C:\app.exe" -Action Block Create a new firewall rule to block an app. (Run as Admin)
Invoke-WebRequest -Uri https://google.com Make a web request. (Alias: iwr, curl)
Invoke-RestMethod -Uri "api.example.com/data" Make a web request to an API, automatically parsing JSON.

System & Process Management

Command Example Description Action
Get-Process | Sort-Object -Property CPU -Descending List processes sorted by CPU usage. (Alias: gps)
Stop-Process -Name "notepad" -Force Forcefully terminate a process by its name. (Alias: spps)
Get-Service -Name "Spooler" Get information about a specific service. (Alias: gsv)
Start-Service -Name "Spooler" Start a service.
Set-Service -Name "MyService" -StartupType DelayedAutomatic Change a service's startup type. (Run as Admin)
Get-ScheduledTask List all scheduled tasks on the system.
Register-ScheduledTask -Action (New-ScheduledTaskAction -Execute "notepad.exe") -Trigger (New-ScheduledTaskTrigger -Daily -At 3am) -TaskName "MyTask" Create a new scheduled task. (Run as Admin)
Get-CimInstance -ClassName Win32_BIOS Get system information (e.g., BIOS). Replaces wmic.
Get-WmiObject -Class Win32_Product | Select Name, Version List installed software (older, slow. Get-CimInstance is newer).
Get-WinEvent -LogName System -MaxEvents 10 Get the last 10 events from the System event log.
Get-EventLog -LogName Application -Newest 10 Get last 10 events from Application log (older, easier syntax).
Start-Transcript -Path C:\Logs\MySession.log Start recording all console commands and output to a log file.
Stop-Transcript Stop recording the console session.
Start-Process "notepad.exe" -Verb RunAs Start a process as an Administrator.
Get-Help Get-Process -Full Get detailed help for any command. (Alias: help, man)
Restart-Computer -Force Restart the computer immediately.

User Management (Run as Admin)

Command Example Description Action
Get-LocalUser List local user accounts.
New-LocalUser -Name "NewUser" -PasswordNeverExpires Create a new local user (will prompt for password).
Remove-LocalUser -Name "OldUser" Delete a local user.
Get-LocalGroupMember -Group "Administrators" List all members of a local group.
Add-LocalGroupMember -Group "Administrators" -Member "NewUser" Add a user to the local Administrators group.
New-LocalGroup -Name "MyGroup" Create a new local group.

Active Directory (Requires RSAT-AD-Tools)

Command Example Description Action
Get-ADUser -Filter 'Name -like "John*"' -Properties * Find all AD users whose names start with "John" and show all properties.
Set-ADUser -Identity "jdoe" -Description "New Role" Update the description for the user "jdoe".
Unlock-ADAccount -Identity "jdoe" Unlock a locked Active Directory user account.
Get-ADComputer -Filter 'OperatingSystem -like "*Server*"' Find all computer objects running a Server OS.
Get-ADGroupMember -Identity "Domain Admins" -Recursive Get all members of a group, including members of nested groups.
Add-ADGroupMember -Identity "MyGroup" -Members "jdoe" Add user "jdoe" to "MyGroup".

Registry Management (Run as Admin)

Command Example Description Action
Get-Item -Path "HKLM:\Software\Microsoft\Windows" List subkeys in a registry key.
Get-ItemProperty -Path "HKCU:\Software\MyApp" -Name "Version" Get a specific value from a registry key.
New-Item -Path "HKCU:\Software\NewApp" Create a new registry key.
Set-ItemProperty -Path "HKCU:\Software\NewApp" -Name "Setting" -Value 123 Create or update a specific value in a key.
Remove-Item -Path "HKCU:\Software\NewApp" -Recurse Delete a registry key and its subkeys.
Test-Path -Path "HKCU:\Software\NewApp" Check if a registry key exists (returns $true or $false).

Remote Management (PS Remoting)

Command Example Description Action
Enable-PSRemoting -Force Run once on remote PCs to allow remoting. (Run as Admin)
Enter-PSSession -ComputerName "RemotePC" Start an interactive session on a remote computer.
Invoke-Command -ComputerName "RemotePC" -ScriptBlock { Get-Process } Run a single command or script block on a remote PC.
$session = New-PSSession -ComputerName "Server1", "Server2" Create a persistent session to one or more remote PCs.
Invoke-Command -Session $session -ScriptBlock { Get-Service "Spooler" } Run a command against all PCs in the persistent session.
Remove-PSSession -Session $session Close and remove the persistent session.

Scripting Logic

Command Example Description Action
$myVar = "Hello"; Write-Host $myVar Set and use a variable.
if ($myVar -eq "Hello") { Write-Host "Match" } Simple If statement. (-eq, -ne, -gt, -lt, -like)
Get-Process | ForEach-Object { Write-Host $_.Name } Loop over a collection of items. (Alias: foreach, %)
try { 1/0 } catch { Write-Warning "Error!" } Basic error handling block.
function Get-Greeting { param($Name) "Hello, $Name" } Define a simple function with one parameter.
Get-Greeting -Name "World" Call the custom function defined above.
